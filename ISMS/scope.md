# Scope ISMS – LabSec Solutions

ISMS se vztahuje na Ubuntu Server, Ubuntu Desktop a lokální síť LAN1 v testovacím lab prostředí provozovaném pod názvem LabSec Solutions s.r.o.  
Rozsah zahrnuje:

- Správu síťových adaptérů a konfigurací (NAT + LAN1)
- Uživatelské účty a přístupová práva
- Konfigurační soubory a logy
- Firewall a bezpečnostní pravidla
- Provozní zálohy a instalovaný software

Cílem je zajistit důvěrnost, integritu a dostupnost systémů a dat v rámci tohoto labu.
